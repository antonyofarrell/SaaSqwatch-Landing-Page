# SaaSqwatch Site — UI Kit

A high-fidelity interactive recreation of the SaaSqwatch editorial / review site: a video review channel tracking the best and worst SaaS/AI tools.

## What's in here

- `index.html` — interactive click-through. Home feed → review detail → tier list.
- `App.jsx` — top-level router / demo shell.
- `TopNav.jsx` — fixed nav with search and subscribe.
- `HeroSpotlight.jsx` — featured episode hero with teal spotlight.
- `ReviewCard.jsx` — video card with tier chip, thumbnail, meta.
- `ReviewGrid.jsx` — responsive grid layout.
- `TierList.jsx` — S/A/B/C/D/F tier rack.
- `VerdictPanel.jsx` — pros/cons/score panel.
- `Footer.jsx` — minimal brand footer.

## Why these components

The site's job is: (1) pull you into a featured review, (2) let you browse recent tests, (3) let you skim verdicts fast (tier list + verdict chips). The components above cover those three jobs at pixel-fidelity to the brand.
